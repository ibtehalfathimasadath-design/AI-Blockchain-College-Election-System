How It Works

Our 3-step automated process ensures fair and secure elections

1
📋

Nomination Verification

📄 Upload ID Card, Fee Receipt & Result Sheet
🔍 AI OCR extracts and validates document data
✅ System auto-checks eligibility (year, result pass)
📧 Automated email confirmation sent
2
🗳️

Secure Voting

🪪 Voter uploads ID Card for OCR verification
📷 Live Iris compared with ID card photo
🗳️ Identity verified → Vote cast
⛓️ Vote stored on blockchain permanently
3
🏆

Transparent Results

📊 Admin monitors live vote count dashboard
🔗 All votes verifiable via blockchain hash
🏆 Winner announced with one click
📧 Auto email sent to all participants

Technology Stack

Built with cutting-edge AI and security technologies

🤖

EasyOCR

Extracts text from ID cards, fee receipts, and result sheets with high accuracy

👁️

Iris Recognition

Compares ID card face with live Iris to prevent impersonation

⛓️

Blockchain

SHA-256 chained blocks store votes immutably. Any tampering is instantly detected

🐍

Python Flask

Lightweight backend with SQLite database for fast and reliable performance

📧

SMTP Email

Automated email notifications for nominations results and winner announcements

🔐

Secure Auth

Werkzeug password hashing + Flask-Login session management
